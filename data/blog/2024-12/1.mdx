### 1.2 Hello World

**内容**

- 编写一个"Hello World"程序。

**与 JavaScript 的区别**

- 使用 `rustc` 命令编译（用户不再需要安装 Rust 环境）。
- 编译和运行是分开的步骤。

**`!` 的含义**

- `!` 表示调用一个宏（macro）。宏与函数（`fn`）不同。

### 1.3 Cargo

**内容**

- Cargo 是 Rust 的包管理工具，类似于 JavaScript 的 Webpack。

**如何使用**

- 创建项目：`cargo new hello_cargo`
- 构建项目：`cargo build`
- 运行项目：`cargo run`
- 检查代码：`cargo check`

**build、run 和 check 的区别**

- `run` = 构建并运行目标文件。
- `check`：不生成可执行文件，仅检查代码是否正确。
- `build --release`：为发布版构建代码。

### 2 猜数字游戏

**内容**

- 游戏生成一个 1-100 的随机数，用户猜测该数字。

**如何获取用户输入**

- 使用：`std::io::stdin().read_line(&mut guess).expect("Failed to read line");`

**如何将字符串转换为数字**

- 使用：`let guess: i32 = guess.trim().parse().expect("Please type a number!");`

**如何生成随机数**

- 使用 `rand::Rng`：
  ```rust
  let secret_number = rand::thread_rng().gen_range(1, 101);
  ```

**如何比较两个数字**

- 使用 `match` 语句：
  ```rust
  match guess.cmp(&secret_number) {
      Ordering::Less => println!("Too small!"),
  }
  ```
- `cmp` 返回一个枚举类型 `Ordering`，有三个值：`Less`、`Greater`、`Equal`（可以使用 `match` 根据不同值返回不同结果）。

**如何创建一个循环**

- 使用 `loop` 语句：
  ```rust
  loop {}
  ```
